# Getting and Cleaning Data - Course Project

This repository contains the R code and documentation for the UCI HAR Dataset project.

### Files
* `run_analysis.R`: The script that performs the 5 steps required by the assignment.
* `CodeBook.md`: Describes the variables, the data, and the transformations performed.
* `TidyDataest_Jaafar_Reshy.txt`: The final output file.

### Folders
* `raw data`: The raw dataset downloaded from the website (https://archive.ics.uci.edu/dataset/240/human+activity+recognition+using+smartphones)

### How to use
1. Download the data from: https://archive.ics.uci.edu/static/public/240/human+activity+recognition+using+smartphones.zip
2. Unzip the folder.
3. Place `run_analysis.R` in the same directory as the `UCI HAR Dataset` folder.
4. Run the script in RStudio.
